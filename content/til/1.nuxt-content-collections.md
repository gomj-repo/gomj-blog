# Nuxt Content v3 Collections

## Collections란?
Nuxt Content v3에서 Collections는 콘텐츠를 타입별로 분류하고 스키마를 정의하는 방법입니다.

## 정의 방법

`content.config.ts`에서 Collections를 정의합니다:

```typescript
import { defineContentConfig, defineCollection, z } from '@nuxt/content'

export default defineContentConfig({
  collections: {
    projects: defineCollection({
      type: 'page',
      source: 'projects/**',
      schema: z.object({
        title: z.string(),
        description: z.string().optional(),
        date: z.string().optional(),
      }),
    }),
  },
})
```

## 쿼리 방법

```typescript
const { data } = await useAsyncData('projects', () =>
  queryCollection('projects').order('date', 'DESC').all()
)
```

## 핵심 포인트
- `source`로 디렉토리 매핑
- `schema`로 frontmatter 타입 안전성 확보
- `queryCollection()`으로 타입 안전한 쿼리
