# 포트폴리오 웹사이트

## 개요
Nuxt 4와 Nuxt Content v3를 활용하여 Confluence 스타일의 개발 포트폴리오 사이트를 구축했습니다.

## 기술 스택
- **Frontend**: Nuxt 4, Vue 3, TypeScript
- **Content**: Nuxt Content v3 (Markdown 기반)
- **Styling**: Pure CSS with Custom Properties

## 주요 기능
- Confluence 스타일 사이드바 네비게이션
- 카테고리별 게시판 (Projects, TIL, Notes, Resume)
- 반응형 디자인 (모바일 지원)
- Markdown 기반 콘텐츠 관리

## 아키텍처
- **Base 컴포넌트 패턴**: BaseCard, BaseList, BaseBadge
- **Composable 패턴**: useNavigation, useBoardMeta
- **슬롯 기반 확장**: 재사용 가능한 컴포넌트 설계
